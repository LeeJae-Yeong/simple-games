# Simple Games Hub

Vue 3 + Vite 로 만든 미니 아케이드입니다. 2048 규칙을 기반으로 한 여러 보드 변형(클래식 4×4, 미니 3×3, 파워 러쉬 5×5)을 한 프로젝트 안에서 즐길 수 있습니다.  
각 모드는 독립적인 목표 타일과 등장 확률, 최고 점수를 가지고 있으므로 마음에 드는 모드를 골라 연습해 보세요.

## 개발용 스크립트

```bash
# 패키지 설치
npm install

# 개발 서버 (http://localhost:5173)
npm run dev

# 정적 빌드
npm run build

# 프리뷰 서버
npm run preview
```

## 주요 기능

- 반응형 레이아웃과 터치/포인터 스와이프 입력 지원
- 모드별 설명 및 팁, 최고 점수 로컬 저장
- 간결한 Vue 3 `<script setup>` 구성과 순수 CSS 스타일
- Vue Router 기반 멀티 페이지 구조 (홈 > 게임 선택 > 각 게임)
